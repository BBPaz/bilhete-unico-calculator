# Calculadora Bilhete Único SP

Aplicativo simples, pra calcular diferenças de custo entre vários tipos de recarga (avulso, diário, semanal, mensal) do bilhete único comum em São Paulo.

## Por que fiz isso

Fiz uns cálculos de guardanapo e imaginei que como meu trajeto tinha várias etapas (ônibus + trilho + ônibus, ida e volta todo dia), a recarga mensal compensaria mais do que o avulso. Mas não seria tanta diferença, e dependeria de coisas como saídas no fim de semana e transportes inesperados.

Daí quis transformar isso numa ferramenta de uso simples, pra qualquer um que estiver ponderando a mesma coisa, possa chegar numa resposta rápida.

## Como usar

1. Abra o `index.html` no navegador
2. Adicione seus trajetos (cada trajeto = um grupo de viagens dentro de uma janela de 3h)
3. Selecione quantos dias por semana você faz esse percurso
4. Marque se inclui domingos (ônibus gratuito) e uso extra nos fins de semana
5. Veja a comparação em tempo real

## Valores

Tarifas do bilhete único comum, vigentes a partir de janeiro/2026, consultadas em [sptrans.com.br/tarifas](https://www.sptrans.com.br/tarifas).
